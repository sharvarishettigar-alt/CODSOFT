# 📒 Contact Book

A GUI-based contact manager built with Python and Tkinter. Store, search, view, and delete contacts — all in one simple window.

## ✨ Features

- Add contacts with Name, Phone, Email and Address
- Search contacts by name or phone number
- View full contact details in a popup
- Delete contacts with confirmation prompt
- Live contact list display

---

## 🚀 How to Run

```bash
cd contact-book
python contact.py
```

> Requires Python 3 with Tkinter (comes pre-installed with Python)
