# 🧮 Calculator

A clean, GUI-based calculator built with Python and Tkinter. Supports the four basic arithmetic operations with a simple two-number input interface.

## ✨ Features

- Addition, Subtraction, Multiplication, Division
- Input validation with error alerts
- Division by zero protection
- Simple and clean Tkinter GUI

---

## 🚀 How to Run

```bash
cd calculator
python calc.py
```

> Requires Python 3 with Tkinter (comes pre-installed with Python)
