# ✅ To-Do List

A GUI-based task manager built with Python and Tkinter. Add, complete, and delete your daily tasks with ease.

## ✨ Features

- Add tasks instantly
- Mark tasks as Done
- Delete tasks with confirmation
- Live task list with status indicator (`Yes` / `No`)
- Clean and minimal interface

---

## 🚀 How to Run

```bash
cd todo_list
python list.py
```

> Requires Python 3 with Tkinter (comes pre-installed with Python)
